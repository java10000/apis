用户中心API

简要描述

此类服务由UserCenter项目提供。主要提供针对用户的增删改查功能！

数据结构

ApplicationInfo 应用信息表
DepartmentInfo 部门表
PermissionDefine 权限定义表
PermissionMini 权限
PermissionRegulation 权限规则表
PositionInfo 职位信息表
RoleDefine 角色定义表
RoleMini 角色
RoleRegulation 角色规则表
UserInfo 用户信息表
